Hello, JSP world!

JSP(Java Server Page)란, Java코드(<% %> 또는 <%= %gt;)가 들어가있는 HTML코드를 말하며 서블릿의 복잡함을 좀 더 간단하게 사용할 수 있다.
서블릿보다 쉽고 작성하기 빠른 것이 가장 큰 장점이며 디자인 부분 html과 로직 부분 java 분리시킬 수 있다.

EL

EL(Expression Language)는 '<%= %>' 또는 'out.print()'와 마찬가지로 값을 출력하고자 할 때 사용한다. 사용 문법은 '${}'이며 EL안에서 간단한 연산도 가능하다. 데이터를 화면에 출력하는 것 뿐만 아니라 커스텀 액션 태그에서의 조건절이나 표현절로 사용되기도 한다. 만일 없는 데이터에 대해 표현하게 된다면 null이 아니라 비어있는 값을 출력한다.

Servlet - 2

POST방식은 url에 변수(데이터)를 노출하지않고 요청하는 것으로 보안 유지가 가능하다. 데이터를 body에 포함하여 전송하므로 body의 데이터를 설명하는 content-type헤더필드가 들어가고 어떤 타입인지 명시해줘야한다. body에 데이터가 들어가 전송 길이는 제한이 없지만 최대 요청받는 시간(time out)이 존재해 페이지 요청, 기다리는 시간이 있다. url에 데이터가 노출되지 않아 즐겨찾기나 캐싱이 불가능하다.

액션태그

액션태그는 표준 액션태그와 커스텀 액션태그로 나뉜다. 표준 액션 태그는 JSP에서 기본으로 제공하여 JSP페이지 내에서 바로 사용이 가능하고 태그 앞에 접두어로 jsp:이 붙는다. 커스텀 액션 태그를 사용하기 위해선 별도의 라이브러리가 필요하며 접두어는 사용자가 지정한대로 쓸 수 있다. core, format, functions등 로직에 필요한 태그들로 구성되어있다.
